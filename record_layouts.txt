Exhibit "A" - RECORD LAYOUTS OF VIRGINIA 
CORPORATE, LIMITED PARTNERSHIP & LIMITED LIABILITY COMPANY 
INFORMATION. MVS/DCB Information - Recfm=VB,Lrecl=677,Blksize=17550 
(NOTE: Lrecl=677 is required by MVS for Variable Blocked files. The 
LRECL is the longest record length, 673, plus 4 bytes for a total 
LRECL of 677.) 

FILE 1 -- TABLE FILE TOTAL LENGTH = 64
1 TABLE-REC 
  2 REC-TYPE (A2) Record Type = '01' 
  2 TABLE-IDENTIFIER (A2) Id for Specific Table 
  2 TABLE-CODE (A10) Table code 
  2 TABLE-DESC (A50) Value for Table Code 

See Table Identifier "00" for Description of table types. 

FILE 2 -- CORPORATE FILE TOTAL LENGTH = 673
1 CORP-REC 
  2 REC-TYPE (A2) Record Type = '02' 
  2 CORP-ID (A7) Unique number assigned to Corp 
	Pos. 1 F = Incorporated outside VA 
	Pos. 1 Numeric = Incorporated in VA 
  2 CORP-NAME (A100) Name of Corporation 
  2 CORP-STATUS (A2) + Corporation Status 
					   Status '00' '01' '02' '14' 
					   are all considered 'ACTIVE' 
  2 CORP-STATUS-DATE (N8) Date of Last Status Change 
  2 CORP-PER-DUR (N8) Date on which Corp will expire 
  2 CORP-INC-DATE (N8) Date of Incorporation 
  2 CORP-STATE-INC (A2) + State of Incorporation 
  2 CORP-IND-CODE (A2) + Industry Code 
  2 CORP-STREET1 (A45) Principal Street 
  2 CORP-STREET2 (A45) Principal Street 
  2 CORP-CITY (A23) Principal City 
  2 CORP-STATE (A2) Principal State 
  2 CORP-ZIP (N9) Principal Zip 
  2 CORP-PO-EFF-DATE (N8) Effective date of Principal address 
  2 CORP-RA-NAME (A100) Registered Agent 
  2 CORP-RA-STREET1 (A45) RA Street 
  2 CORP-RA-STREET2 (A45) RA Street 
  2 CORP-RA-CITY (A23) RA City 
  2 CORP-RA-STATE (A2) RA State 
  2 CORP-RA-ZIP (N9) RA Zip 
  2 CORP-RA-EFF-DATE (N8) Effective date of RA 
  2 CORP-RA-STATUS (A1) + RA Status 
  2 CORP-RA-LOC (A3) + RA Court Locality 
  2 CORP-STOCK-IND (A1) Stock Indicator 
						Valid Values: N - Non Stock 
						S - Stock 
  2 CORP-TOTAL-SHARES (N11) Total Shares of Stock 
  2 CORP-MERGER-IND (A1) Merger Indicator 
  Valid Values: ' ' - Not in Merger 
				 N - Non-Survivor 
				 S - Survivor 
  2 CORP-ASMT-IND (A1) + Assessment Indicator 
  2 CORP-STOCK-INFO 
	3 CORP-STOCK-CLASS (A8) + Stock Class (Occurs 8 times) 
	3 CORP-STOCK-SHARE-AUTH (N11) Number of Shares (Occurs 8 times)
